## PC1-001 — Sala de Situação como ponto de entrada natural do Deputado

**Data:** 2026-06-26
**Observado por:** Shadow User (CTO)
**Perfil:** Deputado
**Status:** 🔵 Observação — aguardando confirmação por outros perfis

**Contexto:** Sessão de primeiro uso como Deputado navegando naturalmente pela plataforma sem roteiro definido.

**O que aconteceu:** A Sala de Situação tornou-se naturalmente o ponto inicial do dia, enquanto o Dashboard não comunica claramente esse fluxo. O usuário gravitou para `/sala-situacao` como ponto de comando, mas a navegação padrão abre `/dashboard` primeiro.

**Impacto:** Cosmético / UX — não bloqueia operação, mas pode gerar atrito de orientação para usuários reais no primeiro acesso.

**Decisão:** Aguardando confirmação por outros perfis e por sessões adicionais do Deputado. Não implementar ainda.
